üí° My Recommendation

Choose Option A: Dropdown Menus

Option A provides the best balance of simplicity, space efficiency, and user familiarity. It solves all 4 current issues while delivering the maximum screen real estate for web content (+50% in portrait, +120% in landscape). The dropdown pattern is well-understood, requires no gesture learning, and is straightforward to implement.

Option B is great if you want a more modern, Chrome-like experience with visual tab previews, but it requires more development effort and slightly more UI overhead.
