## Quick Start

1. **Install**: AvaLearnLite is included with VoiceOS
2. **Enable**: Grant accessibility permissions to VoiceOS
3. **Open**: Launch "AvaLearnLite" from app drawer
4. **Explore**: Press "Start Exploration" and use target app
5. **Export**: Save learned data with "Export to AVU"
