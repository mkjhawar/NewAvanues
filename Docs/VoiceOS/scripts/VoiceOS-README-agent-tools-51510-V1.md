# Agent Tools Directory

This directory contains utility scripts and tools used for development and maintenance of the VOS4 project.

## Scripts Overview

### Python Scripts
- `analyze_errors.py` - Error analysis and reporting tool

### Shell Scripts
- `analyze_imports.sh` - Import statement analysis
- `enable-accessibility-adb.sh` - ADB accessibility setup
- `fix_warnings.sh` - Warning cleanup automation
- `fix-all-voicecursor-redundancy.sh` - VoiceCursor redundancy fixes
- `fix-voicecursor-redundancy.sh` - Single VoiceCursor redundancy fix
- `organize_imports.sh` - Import organization tool
- `safe_import_cleanup.sh` - Safe import cleanup
- `setup-modules.sh` - Module setup automation
- `targeted_import_cleanup.sh` - Targeted import cleanup

## Usage

These scripts are primarily used by AI agents and developers for:
- Code cleanup and maintenance
- Build automation
- Error analysis
- Import management
- Module setup

## Security Note

All scripts have been reviewed for security. Do not run scripts from untrusted sources.
