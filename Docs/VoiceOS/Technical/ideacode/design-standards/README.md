# Design Standards

Project-specific design standards captured using IDEACODE v8.4 hash-tag system.

## Quick Start

Use inline hash-tags during development:

```
User: Build authentication. #project Use Firebase Authentication
```

The AI will automatically validate, categorize, and save the standard.

## Available Hash-Tags

- `#project` - Project-specific standard (saved here)
- `#global` - Universal standard (saved to framework)
- `#test` - Test case
- `#backlog` - Future work item
- `#security` - Security standard
- `#docs` - Documentation update
