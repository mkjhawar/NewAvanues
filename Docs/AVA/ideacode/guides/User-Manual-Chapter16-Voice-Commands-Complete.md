# User Manual - Chapter 16: Complete Voice Command Reference

## Overview

AVA understands over 100 voice commands across 20+ categories. This chapter provides a complete reference of all supported commands organized by category.

## Tips for Best Results

| Tip | Example |
|-----|---------|
| Be specific | "Call John Smith" not just "Call John" |
| Use natural language | "Set a timer for pasta" works |
| Chain commands | "Turn off lights and lock doors" |
| Use pronouns | "Play it on the TV" after selecting media |
| Correct yourself | "No, I meant tomorrow" |
