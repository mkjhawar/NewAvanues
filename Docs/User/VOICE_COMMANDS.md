# Voice Commands Guide

## Overview

WebAvanue supports hands-free browsing through voice commands. Simply tap the microphone button in the address bar and speak your command. The browser will execute the command immediately.

## Getting Started

### Activating Voice Commands

1. Tap the **microphone icon** in the address bar
2. Wait for the green indicator (listening state)
3. Speak your command clearly
4. The command executes automatically

### Voice Recognition States

- **Blue microphone**: Ready to listen
- **Green microphone**: Currently listening
- **Command bar appears**: Shows visual feedback when voice is active

## Navigation Commands

Control page navigation with voice.

| Command | Action | Example |
|---------|--------|---------|
| **"go back"** | Navigate to previous page | "go back" |
| **"go forward"** | Navigate to next page | "go forward" |
| **"go home"** | Go to home page | "go home" |
| **"refresh"** | Reload current page | "refresh" |
| **"go to [url]"** | Navigate to specific URL | "go to google.com" |

### Examples

- "go back" - Returns to the previous page
- "go to reddit.com" - Opens Reddit
- "refresh" - Reloads the current page

## Scrolling Commands

Scroll through pages without touching the screen.

| Command | Action |
|---------|--------|
| **"scroll up"** | Scroll page up |
| **"scroll down"** | Scroll page down |
| **"scroll to top"** | Jump to top of page |
| **"scroll to bottom"** | Jump to bottom of page |
| **"page up"** | Scroll one page up |
| **"page down"** | Scroll one page down |

### Pro Tip

Use "scroll to top" and "scroll to bottom" for quick navigation on long pages.

## Tab Commands

Manage multiple tabs with voice.

| Command | Action |
|---------|--------|
| **"new tab"** | Open a new tab |
| **"close tab"** | Close current tab |
| **"next tab"** | Switch to next tab |
| **"previous tab"** | Switch to previous tab |
| **"reopen tab"** | Reopen last closed tab |

### Multitasking Tips

- Use "new tab" to open links without losing your place
- "close tab" when finished with a page
- Quickly switch between tabs with "next tab" / "previous tab"

## Zoom Commands

Adjust page zoom level by voice.

| Command | Action |
|---------|--------|
| **"zoom in"** | Increase zoom level |
| **"zoom out"** | Decrease zoom level |
| **"reset zoom"** | Reset to 100% zoom |

### When to Use

- **Zoom in**: Make text larger on small screens
- **Zoom out**: See more content at once
- **Reset zoom**: Return to default size

## Tips & Tricks

### Best Practices

1. **Speak clearly**: Enunciate commands for best recognition
2. **Wait for green**: Only speak when the microphone is green
3. **Use exact phrases**: Commands must match exactly (see tables above)
4. **Keep it short**: Simple commands work best

### Troubleshooting

**Voice commands not working?**

1. Check microphone permissions in Android settings
2. Ensure internet connection (voice recognition requires connectivity)
3. Try tapping the microphone again to restart listening
4. Verify voice commands are enabled in Settings → Voice & AI

**Commands not recognized?**

- Speak more slowly
- Reduce background noise
- Use the exact command phrases from this guide
- Check if your language is supported

### Accessibility

Voice commands are perfect for:

- **Hands-free browsing** while driving (use responsibly)
- **Accessibility** for users with motor impairments
- **Multitasking** when your hands are busy
- **AR/XR browsing** with smart glasses
