## Display Settings

Control the visual appearance of web pages.

### Theme
Choose your color scheme.

- **Light**: Bright background (best for daytime)
- **Dark**: Dark background (easy on the eyes at night)
- **System**: Follow device theme
- **Auto**: Changes based on time of day

### Font Size
Adjust text size for readability.

- **Tiny**: 75% (0.75x)
- **Small**: 87.5% (0.875x)
- **Medium**: 100% (default)
- **Large**: 112.5% (1.125x)
- **Huge**: 125% (1.25x)

### Other Display Options

| Setting | Description | Default |
|---------|-------------|---------|
| **Force Zoom** | Allow pinch-to-zoom on all pages | Off |
| **Show Images** | Display images on pages | On |
| **Use Desktop Mode** | Load desktop versions by default | Off |
| **Initial Scale** | Default page zoom level | 75% |

---

## Desktop Mode Settings

Fine-tune desktop mode behavior.

| Setting | Description | Default |
|---------|-------------|---------|
| **Default Zoom** | Zoom level in desktop mode (50-200%) | 100% |
| **Window Width** | Simulated screen width (800-1920px) | 1280px |
| **Window Height** | Simulated screen height (600-1200px) | 800px |
| **Auto-Fit Zoom** | Automatically adjust zoom to fit content | On |

**Tips:**
- Lower zoom for wider view of desktop sites
- Increase window dimensions for more content
- Enable auto-fit zoom for optimal viewing in landscape

## Privacy Settings

Protect your privacy while browsing.

### Content Blocking

| Setting | Description | Default |
|---------|-------------|---------|
| **Block Popups** | Prevent popup windows | On |
| **Block Ads** | Block advertising content | On |
| **Block Trackers** | Block tracking scripts | On |
| **Do Not Track** | Send DNT header to websites | On |

### Data Clearing

| Setting | Description | Default |
|---------|-------------|---------|
| **Clear Cache on Exit** | Delete cached files when closing | Off |
| **Clear History on Exit** | Delete browsing history when closing | Off |
| **Clear Cookies on Exit** | Delete cookies when closing | Off |

### Web Technologies

| Setting | Description | Default |
|---------|-------------|---------|
| **Enable Cookies** | Allow websites to store cookies | On |
| **Enable JavaScript** | Run JavaScript code on pages | On |
| **Enable WebRTC** | Enable real-time communications | Off |

**Privacy Preset:** Enable all blocking and clearing for maximum privacy.

## WebXR Settings

Configure WebXR (virtual/augmented reality) features.

### Master Controls

| Setting | Description | Default |
|---------|-------------|---------|
| **Enable WebXR** | Allow WebXR content | On |
| **Enable AR** | Allow augmented reality sessions | On |
| **Enable VR** | Allow virtual reality sessions | On |

### Performance

| Setting | Description | Default |
|---------|-------------|---------|
| **XR Performance Mode** | Balance quality vs battery | Balanced |
| **Auto-Pause Timeout** | Minutes before auto-pause | 30 min |
| **Show FPS Indicator** | Display frame rate in XR | Off |
| **Require WiFi** | Only allow XR on WiFi | Off |

**Performance Mode Options:**
- **High Quality**: Maximum quality, higher battery drain (90fps target)
- **Balanced**: Good quality and battery life (60fps target)
- **Battery Saver**: Lower quality, saves battery (45fps target)

**Tips:**
- Use Balanced mode for general XR browsing
- Switch to High Quality for immersive experiences
- Enable Battery Saver for extended XR sessions

## Settings Presets

Quickly apply common configurations.

### Default
Balanced settings for most users.

### Privacy Mode
Maximum privacy protection:
- All blockers enabled
- Clear data on exit
- DuckDuckGo search
- WebRTC disabled
- No sync

### Performance Mode
Optimized for speed:
- Hardware acceleration on
- Page preloading on
- Images on
- Data saver off
- Ad blocking on

### Accessibility Mode
Enhanced readability:
- Force zoom on
- Text reflow on
- Large font size

---

## Resetting Settings

To reset all settings to defaults:

1. Go to **Settings → Advanced**
2. Scroll to bottom
3. Tap **Reset to Defaults**
4. Confirm the action

**Warning:** This cannot be undone!
