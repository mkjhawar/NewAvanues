Design Rules
✓ Text labels auto-show under icon-only buttons in ANY app (not just target apps)
✓ Labels derived from: contentDescription → action label → tooltip → cleaned resource ID
✓ Labels display in the active voice locale (en/es/fr/de/hi)
✓ Number badges only appear when user says "numbers on"
✓ Email comma patterns cleaned: extract sender name from Gmail-style contentDescription
✓ No labels on elements that already have visible text (buttons like "Reply", "Forward")
✓ No labels on email body content (links inside HTML emails)
