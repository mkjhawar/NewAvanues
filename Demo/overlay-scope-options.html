How It Works
Archive
Layer 1 — Text Labels: Always visible on target apps. Shows what you can say. Derived from icon metadata (contentDescription, action labels, resource ID).
2Archive
Layer 2 — Numbers: Only when user says "numbers on". Green circles added on top of text labels. Say the number to activate.
✓
Label fixes: Gmail's ", , , Google Cl" cleaned to "Google Cloud". Icons like archive/delete get "Archive", "Delete" from accessibility metadata.
✓
Minimalist scope: Only toolbar icons + list/thread items badged. No badges on email body links or HTML content.
